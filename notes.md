# Oboeru｜暗記練習Webアプリ

歌詞・セリフ・プレゼン原稿を効率よく暗記できる無料のWebアプリです。

**→ https://oboeru.netlify.app/**

---

## 機能

| モード | 説明 |
|---|---|
| そのまま | テキスト全文を表示して確認 |
| 単語隠し | 単語・助詞単位に隠してクリックで開示 |
| 頭文字 | 各行の先頭1文字だけ表示 |
| 一文隠し | 文単位で隠してクリックで開示 |
| タイピングテスト | 順番入力 / 自由入力の2サブモード |

- `[サビ]` `[1番]` などのセクション区切りで部分練習が可能
- 目次パネルから任意のセクションへジャンプ
- 4種カラーテーマ（パープル・ブルー・グリーン・ライト）
- スマホ対応（サイドバースワイプ開閉・スワイプ削除）
- データはブラウザのlocalStorageに保存（登録不要・サーバーなし）

## 技術スタック

- HTML / CSS / JavaScript（フレームワークなし、3ファイル構成）
- `Intl.Segmenter` API（日本語の単語分割）
- Netlify（ホスティング）

## ファイル構成

```
index.html   HTMLレイアウト・ボタン定義
style.css    全スタイル・テーマ定義
app.js       全ロジック（localStorage・モード切替・テストモード）
```
